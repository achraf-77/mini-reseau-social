# Mini Réseau Social (React + Redux Toolkit + JSON Server)

## Sujet
Application mini réseau social permettant :
- Auth (login simple + sauvegarde dans localStorage)
- Publications : afficher / ajouter / modifier / supprimer
- Likes : liker / annuler like + compteur (1 like max par utilisateur)
- Commentaires : afficher / ajouter + association à un utilisateur
- Filtres : filtrer par hashtag et par auteur (via selectors Redux)
- Bonus : tri par popularité (likes) + protection des routes
